# v13.1 — フロント分割 Phase 1 (土台のみ)

このパッチは `MarketMonitor.jsx` には**一切手を加えない**。
新しいファイルを `src/` 配下に追加するだけのリスクゼロ変更。

## 含まれる変更

### vite.config.js (修正)
- `@/` → `src/` のエイリアスを追加
- `node:path` と `node:url` を追加 import (CommonJS と ESM 両対応の書き方)

├── theme.js                              # PALETTE / FONT_MONO / CHART_UNIVERSE_LABELS

## 動作の不変性

この時点では **`MarketMonitor.jsx` はこれらの新ファイルを import していない**ので、
ビルド出力に違いは出ない (Vite の tree-shaking で未使用ファイルは除外される)。

つまりこの zip を適用 & push しても、本番サイトの見た目・挙動は **完全に同じ**。
v13.1.0 として安全に push できる。

## 次のステップ (v13.1.1 — まだ着手していない)

`MarketMonitor.jsx` の冒頭 (1〜170 行目あたり) のインライン定義を削除し、
`@/utils` `@/theme` `@/components/common` から import する形に書き換える。

その変更は別の zip で出す。先に v13.1 を本番に push してビルドが通ることを確認してから着手するのが安全。
